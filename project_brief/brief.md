Vibe Coding in VS Code
INITIAL PROMPT FOR AGENT
You are acting as both a Django developer and a patient programming instructor. Help me
build a beginner-friendly Django project called Campus SkillSwap. This is a real-world
student marketplace where users can post skills or services they offer to others.
Your job is to:

- generate correct code
- explain what each part does in simple terms
- avoid unnecessary complexity
- help me build the project step by step
- include comments in the code
- warn me about common beginner mistakes
  The app must include:
- user registration, login, logout
- Skill model connected to Django User
- forms to create and edit skill posts
- views for list, detail, create, update, delete, and dashboard
- Bootstrap templates with template inheritance
- Django messages if practical
- admin registration
- SQLite database
  Fields for each Skill post should include:
- title
- description
- category
- price or free option
- contact preference
- availability status
- created_at
- owner
  Enhancements

1. Add a simple beginner-friendly search feature so users can search skills by title or
   category.
2. Review and Rating System - Users should be able to rate and review skills they’ve
   used.
3. Booking / Requests System - Users should be able to request a session for a skill.
4. Make the website your own, add color, styling (CSS), images, whatever you want.
